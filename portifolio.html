Trabalho acadêmico - Banco de Dados Relacional

https://github.com/LeviTars/Trabalho-BancoDeDados-Uninter

Como parte de um projeto acadêmico, desenvolvi dois bancos de dados completos, acompanhados de seus respectivos Modelos Entidade-Relacionamento (MER).

  • Locadora de Veículos: Estruturei um banco de dados para gerenciar o processo de locação, incluindo cadastro de clientes, veículos, contratos e histórico de aluguéis.
  • Rede de Hotéis: Modelei um sistema de reservas para uma rede hoteleira, contemplando funcionalidades como cadastro de hóspedes, disponibilidade de quartos, reservas e pagamentos.

Esses projetos demonstram minha capacidade de análise de requisitos, modelagem de dados e aplicação de boas práticas em banco de dados relacional.



Trabalho acadêmico - Python

https://github.com/LeviTars/Trabalho-python-Uninter

Como parte de minha formação acadêmica, desenvolvi quatro sistemas básicos utilizando Python, com foco em lógica de programação, automação de processos e simulação de cenários do mundo real:

  • Sistema de Descontos para Loja: Simula a aplicação de descontos com base em regras comerciais, como percentual por categoria de produto ou valor total da compra.
  • Sistema para Sorveteria: Permite o registro de pedidos, cálculo de preços com adicionais e geração de resumo do pedido.
  • Sistema para Loja de Impressões: Automatiza o cálculo de custos com base em tipo de impressão, quantidade e acabamento.
  • Sistema de Controle de Funcionários: Gerencia dados de colaboradores, como cadastro, cargos, salários e controle de ponto.

Esses projetos reforçaram minha habilidade em desenvolver soluções práticas com Python, utilizando estruturas de decisão, laços de repetição, manipulação de dados e interação com o usuário via terminal.



Trabalho acadêmico - Desenvolvimento Web

https://github.com/LeviTars/Trabalho-desenvolvimentoWeb-Uninter

Como parte de um projeto acadêmico, desenvolvi um site de portfólio pessoal utilizando HTML, CSS e JavaScript. O objetivo foi criar uma plataforma responsiva e funcional para apresentar meus projetos e habilidades na área de desenvolvimento web.

  • Estrutura semântica com HTML5;
  • Estilização moderna com CSS3, incluindo responsividade para diferentes dispositivos;
  • Interatividade com JavaScript, como menus dinâmicos e animações suaves;

Esse projeto reforçou meus conhecimentos em front-end e boas práticas de desenvolvimento web, além de demonstrar minha capacidade de criar interfaces intuitivas e visualmente agradáveis.
